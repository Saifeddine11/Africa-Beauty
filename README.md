# Africa Beauty

Site vitrine one-page React/Vite pour Africa Beauty.

## Structure

```txt
src/
  App.jsx                 # Assemble les sections du site
  main.jsx                # Point d'entree React
  components/
    common/               # Composants reutilisables
      SectionTitle.jsx
    layout/               # Structure globale
      Header.jsx
      Footer.jsx
  sections/               # Sections principales de la page
    Hero.jsx
    Services.jsx
    About.jsx
    Gallery.jsx
    Contact.jsx
  data/
    siteContent.js        # Textes, navigation, services, galerie
  styles/
    global.css            # Styles globaux
```

## Commandes

```bash
npm run dev
npm run build
npm run preview
```
